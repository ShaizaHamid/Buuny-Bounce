# 🐰 Bunny Bounce

A fun bunny-themed game built with **GDevelop** and exported as an HTML5 web game.

## 🎮 Play Now

👉 **[Play Bunny Bounce](https://buuny-bounce.vercel.app/)**

## 🛠️ Built With

- [GDevelop](https://gdevelop.io/) — no-code/low-code game engine
- HTML5 / JavaScript export
- Hosted on [Vercel](https://vercel.com/)

## 📂 About This Repo

This repository contains the full HTML5 export of the game, including:
- Game logic (JS files generated by GDevelop)
- Sprites and animations (Bunny run, jump, idle, hit, etc.)
- Sound effects
- Fonts and UI assets

## 🚀 Running Locally

Since this is a static HTML5 export, you can run it locally with any static server. For example:

```bash
npx serve .
```

Then open the local URL shown in your terminal.

## 📦 Deployment

This project is deployed on **Vercel** as a static site. Any changes pushed to the `main` branch will automatically redeploy.
